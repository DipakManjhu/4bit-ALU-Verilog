# 4-bit ALU - Verilog HDL

A 4-bit Arithmetic Logic Unit designed and simulated using Verilog HDL.

## Operations Supported
| Opcode | Operation |
|--------|-----------|
| 000    | ADD       |
| 001    | SUB       |
| 010    | AND       |
| 011    | OR        |
| 100    | XOR       |
| 101    | NOT A     |

## Tools Used
- EDA Playground (Icarus Verilog)
- EPWave for waveform viewing

## Files
- alu_4bit.v — ALU design module
- alu_4bit_tb.v — Testbench for simulation
